# SCP-CS :: Doors-Manager-Panel

**Плагин MapPanel** - это компонент административной панели для управления игровыми зонами, дверями и лифтами в рамках системы плагина DELTAxEX. Позволяет администраторам быстро находить локации, управлять доступом и состоянием объектов через более удобный интерфейс.
