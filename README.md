# DeltaEx Admin Panel: MapPanel


[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Описание

`MapPanel` — плагин для административной панели в системе DeltaEx, позволяющий управлять игровыми зонами, дверьми и лифтами через удобный UI.

## Возможности

- **Поиск комнат** по названию (поле ввода + кнопка «Найти»).
- **Управление дверьми** выбранной комнаты:
  - блокировка/разблокировка;
  - открытие/закрытие;
  - разрушение.
- **Управление лифтами**:
  - блокировка/разблокировка использования;
  - активация лифта.
- **Быстрый доступ к зонам** через кнопки-переключатели (LCZ, HCZ, EZ, SPECIAL).
- **Динамическое отображение** комнат выбранной зоны в виде сетки кнопок.

## Интерфейс

1. **Панель поиска**  
   - Поле ввода `roomNameInput` для названия комнаты.
   - Кнопка «Найти» → вызывает `FindRoom()`.

2. **Блок управления дверьми**  
   Кнопки: «Заблокировать», «Разблокировать», «Открыть», «Закрыть», «Взорвать».  
   Каждая вызывает соответствующий метод (`LockDoors()`, `UnlockDoors()` и т.д.).

3. **Блок управления лифтами**  
   Кнопки: «Блокировать», «Разблокировать», «Использовать».  
   Методы: `BlockElevator()`, `UnblockElevator()`, `UseElevator()`.

4. **Зоны быстрого доступа**  
   - Кнопки: LCZ, HCZ, EZ, SPECIAL.
   - При нажатии отображаются комнаты зоны (через `ShowZoneRooms()`).
